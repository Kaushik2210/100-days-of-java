# Java Curriculum — Basics to Expert (50 Days)

Each day has: a notes file in `notes/dayNN-slug.md` (the lesson) and a runnable program in `src/dayNN/ClassName.java` (the code). Progress is tracked in `PROGRESS.md`.

## Phase 1 — Foundations (Days 1–10)
1. Introduction to Java & Environment Setup
2. Variables, Data Types & Type Casting
3. Operators & Expressions
4. Control Flow: if/else & switch
5. Loops: for, while, do-while
6. Arrays & Multidimensional Arrays
7. Strings & the String Pool
8. Methods & Method Overloading
9. Introduction to OOP: Classes & Objects
10. Constructors & the `this` Keyword

## Phase 2 — Object-Oriented Programming (Days 11–20)
11. Inheritance
12. Polymorphism (Compile-time & Runtime)
13. Abstraction: Abstract Classes
14. Interfaces & Default Methods
15. Encapsulation & Access Modifiers
16. Static vs Instance Members
17. The Object Class: equals(), hashCode(), toString()
18. Packages & Access Control
19. Enums
20. Nested & Inner Classes

## Phase 3 — Core APIs & Error Handling (Days 21–28)
21. Exception Handling: try/catch/finally
22. Custom Exceptions & Exception Chaining
23. Collections Framework: List (ArrayList, LinkedList)
24. Collections Framework: Set (HashSet, TreeSet)
25. Collections Framework: Map (HashMap, TreeMap)
26. Iterators, Comparable & Comparator
27. Generics
28. Autoboxing, Wrapper Classes & Immutability

## Phase 4 — Modern Java (Days 29–35)
29. Lambda Expressions & Functional Interfaces
30. Streams API: map/filter/reduce
31. Streams API: Collectors & Grouping
32. Optional & Null Safety
33. Date & Time API (java.time)
34. File I/O: java.io & java.nio
35. Serialization

## Phase 5 — Concurrency (Days 36–41)
36. Threads & the Runnable Interface
37. Synchronization & Locks
38. Executors & Thread Pools
39. CompletableFuture & Async Programming
40. Concurrent Collections
41. Atomic Variables & the Java Memory Model (volatile)

## Phase 6 — Expert Topics (Days 42–50)
42. JVM Internals: Class Loading & Memory Areas
43. Garbage Collection Algorithms & Tuning
44. Design Patterns: Creational (Singleton, Factory, Builder)
45. Design Patterns: Structural (Adapter, Decorator, Proxy)
46. Design Patterns: Behavioral (Observer, Strategy, Command)
47. JDBC & Database Connectivity
48. Build Tools: Maven & Gradle Fundamentals
49. Unit Testing with JUnit & Mocking
50. Performance Profiling & Best Practices

Rule for the daily automation: pick the next unclaimed day number in order, write real teaching notes and a real compilable/runnable program for that specific topic — never a placeholder or generic line.
